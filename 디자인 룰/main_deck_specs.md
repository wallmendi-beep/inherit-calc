# 🏛️ 상속지분 계산기 메인덱(Main Deck) 상세 명세서 (v4.0)

이 문서는 상속인 입력 카드 및 배분 결과표가 위치하는 '메인 유닛'에 대한 물리적 규격과 스타일 가이드를 정의합니다.

---

## 1. 물리적 규격 및 레이아웃 (Physical Specs)

데스크톱 최적화 및 레이아웃 안정성을 위해 아래의 절대 수치를 고수합니다.

| 항목 | 규격 (Tailwind / CSS) | 비고 |
| :--- | :--- | :--- |
| **본문 고정 너비** | `w-[1080px] min-w-[1080px]` | 가이드 패널 오픈 시에도 유지 (Fixed) |
| **본문 탄력성** | `shrink-0` | 어떠한 상황에서도 너비 축소 금지 |
| **카드 내부 여백** | `p-10` | 쾌적한 데이터 입력 공간 확보 |
| **모서리 곡률** | `rounded-xl` | 정갈한 카드 느낌 유지 |
| **테두리 규격** | `border border-[#e9e9e7]` | 노션 스타일 가느다란 테두리 |

---

## 2. 지분 표시 인쇄 및 가독성 (Share Typography)

수치 데이터의 전문성을 위해 색상 위계를 강화했습니다.

- **색상 토큰**: **`text-[#1e56a0]` (Deep Navy)**
- **텍스트 두께**: `font-black` (가장 두꺼운 수치 표현)
- **표시 위계**:
    - **Header 영역**: `text-[17px]`, 선명한 가독성 확보.
    - **Heir Row (리스트)**: `text-[11px]`, 자동 계산 여부와 상관없이 `#1e56a0` 고정 적용.
    - **통분 기호 (`/`)**: `text-neutral-500` (중간 회색).

---

## 3. 워터마크 배치 공식 (Watermark Placement)

워터마크는 프로젝트의 정체성을 나타내되, UI와 충돌하지 않아야 합니다.

- **위치**: 메인 카드 내부 하단 (`absolute bottom-1 right-8`)
- **스타일**:
    - 서체: Cursive (필기체)
    - 크기: `text-[15px]`
    - 투명도: `opacity-60` (데이터를 방해하지 않는 수준)
    - 각도: `rotate(-1deg)` (자연스러운 필기 느낌)

---

## 4. 인쇄 모드 규격 (Print Specs)

- 인쇄 시 워터마크 숨김: `no-print` 클래스 적용.
- 인쇄 시 테두리 제거: `print:border-none`.
- 인쇄용 컴팩트 모드 적용: `print-compact`.

---
> [!NOTE]
> 본 명세는 전문가용 도구로서의 **'일관된 조작감'**과 **'데이터 선명도'**를 최우선으로 합니다.
