# 📐 스마트 가이드 UI 디자인 명세서 (v4.0)

이 문서는 상속지분 계산기 PRO 전용 스마트 가이드의 '비차단(Non-blocking)' 구현과 콤팩트한 디자인 규격을 정의합니다.

---

## 1. 물리적 규격 및 위치 (Physical Dimensions)

본문 컨텐츠를 방해하지 않으면서 정보를 효율적으로 전달하기 위한 수치입니다.

| 항목 | 규격 (Tailwind / CSS) | 비고 |
| :--- | :--- | :--- |
| **패널 너비** | `310px` | 기존 360px에서 축소 (콤팩트 규격) |
| **시작 위치 (Top)** | `102px` | 메인 헤더 하단에서 30px 이격하여 시작 |
| **전체 높이** | `calc(100vh - 102px)` | 화면 하단까지 꽉 차게 배치 |
| **포지션** | `fixed right-0` | 우측 고정형 사이드 서랍(Drawer) |

---

## 2. 배지 시스템 (Emoji Badge System)

텍스트 중심의 무거운 UI를 탈피하고, 간결한 이모지를 통해 가독성을 높였습니다.

- **Fatal (계산불가/필수)**: `🚫` (레드 배경 대신 무채색 원형 칩 내부에 배치)
- **Warning (주의/확인)**: `⚠️` (앰버 배경 대신 무채색 원형 칩 내부에 배치)
- **Info (참고/안내)**: `ℹ️` (블루 배경 대신 무채색 원형 칩 내부에 배치)

> **원칙**: 배지 내부에 '필수', '확인' 등의 텍스트를 절대 병기하지 않습니다.

---

## 3. 카드 및 섹션 테마 (Neutral Theme)

시각적 피로도를 최소화하기 위해 '무채색(Grayscale)' 테마를 고수합니다.

- **카드 배경**: `bg-neutral-50` (약간 밝은 회색)
- **카드 테두리**: `border-neutral-200`
- **섹션 배경(강조)**: `bg-neutral-100` (무결성 경고 등 강조 시 사용)
- **호버 효과**: `hover:bg-neutral-100`

---

## 4. 인터랙션 및 애니메이션

- **비차단(Non-blocking)**: 가이드 패널이 열려 있어도 메인 본문 영역의 조작이 가능해야 함.
- **트랜지션**: `transition-all duration-300 ease-in-out` 적용으로 부드러운 개폐 효과.

---
> [!IMPORTANT]
> **디자인 철학**: 스마트 가이드는 본문 작업의 보조 도구입니다. 
> 화려한 색상으로 시선을 뺏기보다는, 무채색과 이모지를 활용해 필요한 정보만 정갈하게 전달합니다.
