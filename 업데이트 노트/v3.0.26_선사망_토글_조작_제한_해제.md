# 업데이트 노트 (v3.0.26)

이 버전은 선사망 상속인(자녀 등)의 상태 토글 조작이 불가능하던 문제를 해결하고, 조작의 자유도를 복원한 정밀 수정 버전입니다.

## 🛠️ 주요 수정 사항

### 상속인 목록 (HeirRow)
- **선사망 토글 차단 해제**: 선사망(`predeceased`) 상태로 판별된 상속인의 경우에도 사용자가 자유롭게 토글 스위치를 조작할 수 있도록 클릭 차단 로직을 제거했습니다.
- **UI 비활성화 스타일 제거**: 선사망 상태 시 강제로 적용되던 흐림 효과(`opacity-30`)와 금지 표시(`cursor-not-allowed`)를 제거하여 일반 상속인과 동일한 인터페이스 스타일을 적용했습니다.
