# 업데이트 노트 (v3.0.28)

이 버전은 선사망 상속인 토글 조작 시, 데이터 오류 방지를 위해 **3초간의 경고 노출 후 초기 상태로 자동 원복**되는 동적 인터랙션을 구현하고 디자인 룰과 동기화한 버전입니다.

## 🛠️ 주요 수정 사항

### 1. 상속인 목록 (HeirRow)
- **3초 자동 원복 로직**: 선사망(`predeceased`) 상태인 상속인의 토글을 강제로 켤 경우, `setTimeout`을 통해 3초 후 자동으로 다시 `Off` 상태로 원복되도록 구현했습니다.
- **경고 슬라이드 동적 제어**: 위 타이머와 연동하여 경고 슬라이드가 즉각 노출되었다가, 3초 후 토글이 꺼짐과 동시에 슬라이드 업 애니메이션을 통해 자동으로 사라지도록 처리했습니다.

### 2. 디자인 룰 최신화 (main_display_rules.md)
- **규칙 4.2절 업데이트**: '3초 자동 원복' 인터랙션 조건을 명문화하여 앱의 핵심 UX 가이드로 확정했습니다.
