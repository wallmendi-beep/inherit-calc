## 버그 수정

### 1. 사건 그래프 연결선 위치 오류 (busX)
- **증상**: 김혁조 사건에서 구수명의 연결선이 자녀 김명수 카드에서 나오는 것처럼 보임
- **원인**: `busX = 430` 고정값이 상속인 카드 시작 x(420)보다 안쪽에 있어 버스가 카드 내부에 위치
- **수정**: `busX`를 `buildEventLayout`에서 좌열 우측(`leftColX + cardW`)과 우열 좌측(`heirsX`) 사이 중간값으로 동적 계산 후 layout 객체에 포함
- **파일**: `src/components/TreePanel.jsx`

### 2. 사건 그래프 엣지 지분 뱃지 잔존
- **증상**: 배우자 연결선 위에 지분 분배 뱃지(`EdgeLabel`)가 표시됨 (제거하기로 했었음)
- **수정**: `OrthogonalEdges` 배우자 선에서 `<EdgeLabel>` 제거
- **파일**: `src/components/TreePanel.jsx`

### 3. 구수명 사건에서 자녀가 형제자매로 표시
- **증상**: 배우자(구수명)의 상속사건으로 이동 시 자녀들이 형제자매로 표시됨
- **원인**: `inheritance.js`의 `findGlobalSuccessors` 호출 조건이 배우자 노드도 포함 → 배우자에게 후속 상속인 없을 때 피상속인의 혈족 자녀를 '형제자매'로 자동 배정
- **수정**: `!isSpouseRelation(node.relation)` 조건 추가로 배우자 노드는 글로벌 승계인 자동 탐색 제외
- **파일**: `src/engine/inheritance.js`

### 4. 중복인물 판정 오탐 (여러 계통 상속인)
- **증상**: 여러 계통에서 정상 상속받는 인물(윤종옥, 운우경 등)이 모두 중복인물 경고로 표시됨
- **원인**: `collectDuplicatePersonIds`가 트리 전체 personId 출현 횟수를 세는 방식 → 복수 계통 상속은 정상임에도 오탐
- **수정**: 판정 로직 고도화 — 동일 조상의 여러 배우자 탭 아래 동일 자녀(배우자별 자녀 이중 입력)만 감지하도록 변경
- **파일**: `src/utils/importValidationV2.js`

### 5. 사건 그래프 수직 버스 범위 오류
- **증상**: 자녀들이 서로 연결된 것처럼 보이고, 첫 자녀가 버스와 연결되지 않음
- **원인**: 수직 버스가 `rootCenterY`~마지막 자녀까지만 그려지는데, 첫 자녀 center(63)가 rootCenterY(68)보다 위에 있어 연결 끊김
- **수정**: 버스 범위를 피상속인·배우자·모든 자녀의 Y좌표 전체를 아우르는 `min~max`로 변경
- **파일**: `src/components/TreePanel.jsx`

### 6. smartGuideHelpers 계모자 가이드 법령 시대 판단 오류 (기존 버그)
- **증상**: 1983년 사망 피상속인 사건에서 계모자 관계 가이드가 생성되지 않음
- **원인**: `wifeEstateDate = wifeReferenceNode?.deathDate`로 처의 사망일(2022)을 사용 → 1991 현행법 판정
- **수정**: `wifeEstateDate = node.deathDate || tree.deathDate`로 피상속인(estate 분배일) 기준으로 변경
- **파일**: `src/hooks/smartGuideHelpers.js`
