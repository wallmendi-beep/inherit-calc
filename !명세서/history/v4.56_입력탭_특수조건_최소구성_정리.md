# v4.56 입력 탭 특수조건 최소 구성 정리

## 변경 배경
- 데이터 입력 탭이 사건 해석 결과까지 과도하게 떠안아 혼란을 주고 있었음
- 특수조건 칸에서 사실 입력과 법리 결과가 섞여 있었음

## 이번 변경
- 특수조건 칸에서 계산 계수 및 해석 결과성 배지 제거
- `호주상속/재산상속`, `동일가적/비동일가적`, `선사망`, `대습상속인 없음`만 중심으로 재구성
- 선사망자에 대한 `대습상속인 없음` 버튼을 부모 행 특수조건 칸으로 이동

## 기대 효과
- 입력 담당자가 무엇을 입력해야 하는지 더 분명해짐
- 가감산/배제 해석은 시뮬레이션·계산 상세로 역할 분리
