# v4.39 빈상태 안내와 불러오기 부모해석 통일

## 배경
- 구수명 탭에서 빈상태 안내는 자동분배 대상을 보여주는데, `불러오기` 버튼은 반응하지 않는 문제가 있었다.

## 원인
- 빈상태 안내는 현재 트리 기준 부모(`resolvedParentNode`)를 보는데,
- `불러오기`는 옛 `activeTabObj.parentNode` snapshot을 그대로 보고 있었다.

## 수정 내용
- `src/components/InputPanel.jsx`
  - `handleAutoFill()`도 `parentHeirsForGuide`를 사용하도록 변경
  - 이제 안내와 불러오기가 모두 현재 실제 부모 노드를 공통 기준으로 사용

## 기대 효과
- 안내에 보이는 자동분배 후보와 실제 불러오기 후보 일치
- 구수명 같은 배우자 탭에서도 불러오기 정상화

## 검증
- `npm run build` 통과
- `npx vitest run src/engine/inheritance.test.js` 통과
