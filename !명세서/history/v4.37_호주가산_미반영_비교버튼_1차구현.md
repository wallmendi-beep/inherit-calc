# v4.37 호주가산 미반영 비교버튼 1차 구현

작성일: 2026-04-13

## 이번 업데이트의 목표

실제 호주가산이 적용된 사건에서만 `미반영시 결과보기` 버튼을 제공하고, 호주가산을 빼고 다시 계산한 결과와 현재 결과를 바로 비교할 수 있게 한다.

## 주요 변경

1. 호주가산 미반영 비교 계산 유틸 추가
2. `계산 상세` 탭에 `미반영시 결과보기` 버튼 추가
3. `결과 리포트`, `지분 요약표`에도 변동 상속인 목록 노출
4. 변동 상속인은 파란 글씨 기준을 유지

## 이번 단계의 한계

1. 현재는 사건 전체 기준 비교다.
2. 사람별 `차이만 보기` UI는 아직 없다.
3. 버튼은 `계산 상세` 탭에만 직접 배치했다.

## 검증

- `npm run build` 통과
- `npx vitest run src/engine/inheritance.test.js` 통과
