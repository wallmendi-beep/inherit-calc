# 🏛️ 상속지분 계산기 PRO - Update Note (v4.61)

## 📌 업데이트 개요
- **버전**: v4.61
- **날짜**: 2026-04-14
- **주요 목적**: 가계도 시뮬레이션 관련 UI 장애 요인 제거 및 네비게이션 무결성 강화

## 🚀 주요 변경 사항

### 1. 가계도 시뮬레이션(트리) UI 최적화
- **토글 버튼 위치 조정**: 트리 보기/사건 시뮬레이션 전환 버튼을 화면 **좌측**으로 이동하여 스마트 가이드(서랍)에 가려지는 문제를 근본적으로 해결했습니다.
- **사이드바(가계도 요약) 자동화**: 시뮬레이션 탭 진입 시 좌측 패널이 자동으로 닫히도록 `useEffect` 기반 리액티브 제어를 도입했습니다. (입력 탭 복귀 시 자동 열림)

### 2. 지능형 네비게이션 및 계보 관리
- **탭 전환 경로 최적화**: 가이드 클릭 시 데이터 입력창을 거쳐가는 '번쩍임(Flicker)' 현상을 제거하기 위해 `handleNavigate` 로직을 단일 경로로 정밀화했습니다.
- **트리 자동 확장(Auto-Expand)**: 계보 오류 수정이나 특정 인물 확인 시, 접혀있던 부모 노드들을 자동으로 펼쳐서 해당 인물을 즉시 노출합니다.
- **구조적 오류 시각화**: 자녀 위치에 부모/형제가 입력된 '구조적 오류' 노드에 붉은색 링(Ring)과 배경색 강조를 적용하고 즉시 삭제가 가능하도록 했습니다.

## ⚖️ 법리 및 로직 반영
- **사실(Fact)과 판단(Judgment)의 분리**: 데이터 수정은 입력 탭에서, 가계도상의 논리적/구조적 결함은 시뮬레이션 트리 탭에서 즉시 해결할 수 있도록 사용자 경험을 이원화했습니다.

## 🧪 검증 내역
- 일반 인물 클릭 시 데이터 입력 탭의 해당 인물 탭으로 정상 이동 확인.
- 계보 오류 카드 클릭 시 시뮬레이션 트리의 해당 노드로 자동 스크롤 및 계보 확장 확인.
- 시뮬레이션 탭 진입/이탈 시 좌측 패널 자동 개폐 확인.
