# v4.35 호주선택 UI 및 빈상태안내 정리

작성일: 2026-04-13

## 이번 업데이트의 목표

호주상속 가산 관련 입력 UX와 빈상태 안내를 사건별 땜질에서 벗어나, 탭 기준 규칙으로 정리한다.

## 주요 변경

1. 현재 탭 피상속인 카드에 `호주/비호주` 슬림 토글 추가
2. `원호주상속인` 선택 UI 저장/불러오기 연결
3. `호주`만 체크된 경우도 원호주상속인으로 우선 인정
4. 빈상태 안내를 `getEmptyStateGuide()` 단일 함수 기반으로 재구성
5. 중복되던 상단 안내 제거
6. 안내 박스 색을 그레이톤으로 통일
7. 상속인 추가 우측의 작은 `*` AI 버튼 제거
8. 계산 상세 탭의 깨진 문자열 정리용 `CalcPanelFixed` 연결

## 법리/로직 보강

1. 선사망 배우자는 피대습자가 아니므로 대습상속 버튼과 경로를 열지 않음
2. 사위 대습상속 불가 케이스는 토글 차단 + 슬라이드 경고 유지
3. 구법 아들/남편 계통은 AI 입력의 기계적 `isHoju:false`보다 사용자 확인을 우선함
4. 원호주상속인이 없는 구법 호주 단계에서는 엔진 경고를 남김
