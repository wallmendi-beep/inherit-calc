# v4.57 가이드와 경고등 입력중심 분리

## 변경 배경
- 데이터 입력 탭이 계산 결과 경고까지 함께 떠안아 과도한 노이즈가 발생함
- 새 입력창 구조는 fact/vault 입력 중심으로 재정의되었으므로, 가이드도 그에 맞게 단순화할 필요가 있었음

## 이번 변경
- 입력 탭 가이드에서 감사 엔진 이슈를 직접 통합하던 로직 제거
- 입력 탭에서 raw engine warning 섹션 비활성화
- 감사 엔진 이슈와 repair hints는 시뮬레이션/계산/결과 탭에서만 노출

## 기대 효과
- 입력 담당자가 사실 입력에 집중할 수 있음
- 경고등이 과거처럼 같은 문제를 여러 번 반복하지 않음
- 시뮬레이션/계산 탭이 사건 결과 검산 역할을 더 분명하게 담당함
