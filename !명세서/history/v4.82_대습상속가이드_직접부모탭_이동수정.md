# 업데이트 노트 (v4.82) - 대습상속 가이드 직접 부모 탭 이동 수정

작성일: 2026-05-03

## 수정 배경

`대습상속 미확정 — [구수명] 사건의 윤종옥 계통 선사망자...` 카드를 눌러도 화면 이동이나 하이라이트가 보이지 않는 문제가 있었다.

## 원인

대습상속 미확정 가이드는 하이라이트할 선사망자 `targetNodeIds`를 가지고 있어 데이터 입력 탭으로 이동한다. 그런데 `targetTabId`가 사건 맥락의 피상속인인 구수명으로 잡혀 있었다.

구수명 입력 탭에는 윤우경, 윤우철, 정차윤, 정문자 행이 직접 렌더링되지 않는다. 이 인물들은 윤종옥 또는 김명수 같은 하위 계통 탭의 직접 자녀 행이므로 DOM에서 찾을 수 없어 클릭해도 아무 반응이 없는 것처럼 보였다.

## 반영 내용

- 대습상속 미확정 그룹 키를 `contextNode + 직접 부모` 단위로 변경
- 문구는 `[구수명] 사건의 윤종옥 계통...`처럼 사건 맥락을 유지
- 클릭 이동 대상은 실제 하이라이트 행이 존재하는 직접 부모 탭으로 변경
- 서로 다른 직접 부모 계통의 선사망자는 한 카드에 섞지 않고 별도 카드로 분리

## 기대 결과

- 윤종옥 계통 카드는 윤종옥 탭으로 이동하여 윤우경/윤우철 등 직접 행을 하이라이트한다.
- 김명수 계통 카드는 김명수 탭으로 이동하여 정차윤/정문자 등 직접 행을 하이라이트한다.
- 사용자는 해당 행에서 대습상속인을 입력하거나 `대습상속인 없음`을 확정할 수 있다.
