# 업데이트 노트 (v4.73) - UX 텍스트 경량화 및 디자인 디테일 개선

## 1. 개요
* **날짜**: 2026-04-24
* **주제**: 계산기 UI의 가독성을 높이기 위한 텍스트 간소화 및 시각적 디테일(슬라이더, 워터마크) 개선
* **목적**: 불필요한 조사와 중복 단어를 제거하여 데이터 밀도를 높이고, 인터페이스의 전문가적 느낌(음각, 얇은 선)을 강화하여 사용자 경험을 고도화함.

## 2. 주요 변경 사항

### 2.1. 결과표 취득 경로 텍스트 간소화
* **대상 파일**: `ResultPanel.jsx`, `ResultPanelFixed.jsx`, `SummaryPanelFixed.jsx`, `PrintReport.jsx`
* **변경 내용**: 결과표(요약 및 인쇄본 포함)의 취득 경로 문구에서 불필요한 조사와 서술어 제거. 법령 정보는 시각적으로 분리.
  * *이전*: "망 김영서의 아들으로 1979 개정민법 적용"
  * *개선*: "망 김영서의 아들 <1979년 개정민법>"

### 2.2. 스마트 가이드 문구 개선
* **대상 파일**: `SmartGuidePanel.jsx`
* **변경 내용**: 정상 상태일 때의 안내 문구를 더 직관적이고 스캐너블하게 줄바꿈 적용.
  * *이전*: "정상 / 입력 기준으로 바로 잡아야 할 핵심 문제는 없습니다. / 사건 검토나 계산 상세에서 사건별 변화만 이어서 확인하면 됩니다."
  * *개선*: "이상 없음 / 가계도 검증이 완료되었습니다. / 사건 검토 및 계산 결과를 확인해 주세요."

### 2.3. 사건 그래프(TreePanel) 디자인 개선
* **대상 파일**: `TreePanel.jsx`
* **변경 내용**: 좌측 슬라이드 다이얼(`DialScrubber`)을 얇고 가볍게 수정하여 세련미 강화.
  * 스크롤 트랙 라인: `3px` → `2px`
  * 활성화된 닷(현재 사건): 링 두께 `2px` → `1.5px`, 크기 축소 (`12px` → `10px`)
  * 비활성화된 닷: 크기 축소 (`6px` → `4px`)

### 2.4. 음각 필기체 워터마크 추가
* **대상 파일**: `InputPanel.jsx`
* **변경 내용**: 입력 화면 우측 하단에 `Designed by J.H. LEE` 워터마크 추가.
  * 필기체 폰트 적용 (`Caveat`, `Dancing Script`, `Great Vibes`)
  * `text-shadow`를 활용한 정밀한 음각(Engraved) 효과 구현 (라이트/다크 모드별 별도 컬러 처리)

## 3. UI 디자인 시스템(UDS) 업데이트
* `UI_DESIGN_SYSTEM.md` 문서를 업데이트하여 신규 워터마크 명세와 텍스트 경량화 원칙(취득 경로 축약, 가이드 문구 직관화)을 명문화함.

## 4. 향후 계획 및 검토 사항
* 간소화된 텍스트가 복잡한 재상속 구조에서도 가독성을 충분히 제공하는지 모니터링
* 얇아진 스크롤바와 워터마크가 다양한 화면 해상도에서 의도한 대로 렌더링되는지 확인
