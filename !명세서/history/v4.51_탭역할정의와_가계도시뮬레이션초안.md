# v4.51 탭 역할 정의와 가계도 시뮬레이션 초안

작성일: 2026-04-13

## 변경 목적

기존 입력 탭 중심 구조에서 벗어나,  
`데이터 입력 / 가계도 / 계산 상세 / 결과 탭`의 책임을 다시 분명히 정의한다.

특히 가계도 탭을 단순 트리 요약이 아니라  
**사망자별 지분 이동 시뮬레이션 탭**으로 확장하기 위한 설계 초안을 만든다.

## 이번에 정리한 내용

1. 데이터 입력 탭 역할 재정의
- vault 원본 정리
- 사실관계 입력
- 정적 입력/수정 화면

2. 가계도 탭 역할 재정의
- vault -> tree 구조 확인
- 사망자별 상속 흐름 카드
- 지분 이동 시뮬레이션

3. 계산 상세 탭 역할 재정의
- vault + tree + 엔진 계산 결과 검산
- 규정, 계산식, 상속분, 비고 확인
- 이름 클릭 시 입력 탭 이동

4. 가계도 시뮬레이션 UI 초안 작성
- `트리 보기 / 흐름 보기`
- 사망자별 카드
- 원지분, 1차 분배, 대습/재상속, 최종 귀착 표시
