# v4.42 불러오기 confirm 제거 / 검수 가이드 전환

## 핵심 변경
- AI/파일 불러오기 시 `window.confirm` 차단 흐름 제거
- 선검사 결과를 `importIssues` 상태로 저장
- 검수 필요 항목을 스마트 가이드로 노출
- 검수 항목이 있으면 입력 탭으로 이동하여 즉시 수정 유도
- 수정 후 현재 트리를 다시 검사해 검수 가이드 자동 갱신

## 변경 파일
- `src/utils/importValidationV2.js`
- `src/utils/appActions.js`
- `src/App.jsx`
- `src/hooks/useSmartGuide.js`

## 기대 효과
- 사용자가 무조건 `예`를 눌러야 하는 막힌 흐름 제거
- AI/OCR 오류를 앱 내부에서 바로 수정 가능
- 검수 항목이 사람별 가이드로 남아 추적성과 복구성이 높아짐
