# 업데이트 노트 (v4.91) - AI 자동입력 도움말 명세화

작성일: 2026-05-07

## 1. 배경

앱 도움말 모달에 `AI 입력 방법과 주의사항`을 추가했지만, 실제 사용 흐름과 맞지 않는 표현이 있었다.

문제 표현은 `AI 입력 기능에 자료를 넣어 JSON 초안을 생성한다`는 식의 설명이었다. 실제 기능은 계산기가 사진이나 PDF를 직접 분석하는 방식이 아니라, 계산기 전용 프롬프트를 외부 AI 도구에 넣어 JSON을 만든 뒤 그 JSON을 계산기에 붙여넣는 방식이다.

또한 이 실제 사용 흐름은 프로젝트 루트의 `HELP_MANUAL_DRAFT.md`에는 정리되어 있었지만, `!명세서`의 정식 specs 문서로는 들어가 있지 않았다.

## 2. 문서 정리

새 명세 문서를 추가했다.

- `!명세서/specs/AI_AUTOINPUT_USER_FLOW_2026-05-07.md`

이 문서는 다음을 정식 기준으로 둔다.

- `AI 자동입력`은 계산기가 직접 사진/PDF를 분석하는 기능이 아니다.
- 사용자는 계산기 전용 프롬프트를 복사한다.
- 사용자는 외부 AI 도구에 자료와 전용 프롬프트를 함께 입력한다.
- 외부 AI가 생성한 JSON 전체를 계산기의 `AI 자동입력` 창에 붙여넣는다.
- 앱은 JSON을 정규화하고, 사용자는 입력 탭·스마트 가이드·사건 검토 탭에서 결과를 검수한다.

기존 탭 도움말 문서도 위 흐름에 맞게 수정했다.

- `!명세서/specs/TAB_HELP_CONTENT_2026-05-07.md`

## 3. 앱 도움말 수정

`src/components/TabHelpModal.jsx`의 데이터 입력 도움말 중 `AI 자동입력 방법과 주의사항` 섹션을 수정했다.

반영 내용:

- 계산기가 직접 사진/PDF를 분석하지 않는다는 점을 명시
- 전용 프롬프트 복사 -> 외부 AI 사용 -> JSON 붙여넣기 절차를 단계별로 표시
- AI 결과는 입력 초안이며, 이름·관계·날짜·구법 요소·누락 상속인·구조 오류를 사용자가 검수해야 한다는 점을 보강
- 외부 AI 사용 전 불필요한 개인정보 제거 권고 추가
