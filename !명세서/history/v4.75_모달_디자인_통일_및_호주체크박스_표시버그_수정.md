# 업데이트 노트 (v4.75) - 모달 디자인 통일 및 호주 체크박스 표시 버그 수정

## 1. 개요
- 날짜: 2026-04-29
- 목적:
  1. 세 모달(초기화 확인, AI 자동입력, 사건 검토)의 디자인·문구를 통일
  2. 기본정보 바의 호주 체크박스가 하위 탭 이동 시 체크가 풀려 보이는 표시 버그 수정

## 2. 주요 수정 사항

### 2.1. 모달 디자인 통일 (ResetConfirmModal / AiImportModal / PersonEditModal)

#### 변경 전 불일치 항목
| 항목 | ResetConfirmModal | AiImportModal | PersonEditModal |
|---|---|---|---|
| z-index | z-[9999] | z-[200] | z-[300] |
| 오버레이 | bg-slate-900/55 backdrop-blur-[2px] | bg-black/40 backdrop-blur-sm | bg-black/40 backdrop-blur-sm |
| 카드 radius | rounded-2xl | rounded-xl | rounded-xl |
| 헤더 | 그라디언트 + 영어 "Reset" 라벨 | 일반 헤더 + X버튼 | 일반 헤더 + X버튼 |
| 애니메이션 | 없음 | duration-200 | duration-150 |

#### 수정 내용

**ResetConfirmModal.jsx**
- z-index `z-[9999]` → `z-[200]`
- 오버레이 `bg-slate-900/55 backdrop-blur-[2px]` → `bg-black/40 backdrop-blur-sm`
- 카드 radius `rounded-2xl` → `rounded-xl`
- 그라디언트 헤더 제거, 영어 "Reset" 라벨 삭제
- 헤더에 X버튼 추가
- 애니메이션 `animate-in fade-in duration-150` 추가
- `"그냥 초기화"` → `"저장 없이 초기화"` (더 명확하고 중립적인 표현)
- 푸터 배경색 추가 (`bg-[#f7f7f5]/50`)

**AiImportModal.jsx**
- 애니메이션 `duration-200` → `duration-150` (통일)
- `"직접 입력 실행"` → `"입력 실행"` (더 간결한 표현)

**PersonEditModal.jsx**
- z-index `z-[300]` → `z-[200]`
- `max-w-[456px]` → `max-w-md` (표준 Tailwind 클래스 사용)
- 헤더 `"사건 검토"` → `"{node.name} 검토"` (어떤 인물인지 즉시 확인 가능)
- 하드코딩된 `"정문자"` → `node.name` 동적 참조

### 2.2. 기본정보 호주 체크박스 표시 버그 수정

- 대상 파일: `src/components/InputPanel.jsx`
- 증상:
  - 기본정보 바에서 피상속인의 호주 항목에 체크한 후, 하위 상속인 탭으로 이동하면 체크가 풀려 보임
  - 실제 `tree.isHoju` 값은 바뀌지 않았으나 화면 표시만 `false`로 렌더링됨
- 원인:
  ```jsx
  // 수정 전
  checked={isRootNode ? tree.isHoju !== false : false}
  // isRootNode가 false(하위 탭)이면 무조건 false를 표시
  ```
- 수정:
  ```jsx
  // 수정 후
  checked={tree.isHoju !== false}
  // 항상 실제 tree.isHoju 값을 표시
  ```
  - `disabled={!isRootNode}` 속성은 그대로 유지되므로 루트 탭 외에서는 편집 불가

## 4. 비고

- 호주 체크박스 버그는 상태 변경 없이 렌더링 로직만 수정한 것으로, 데이터에는 영향 없음.
