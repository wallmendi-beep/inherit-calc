# v4.59 대습상속과 재상속 경우의 수 정리

## 요약

- 대습상속과 재상속을 별도 축으로 나눠 경우의 수를 문서화했다.
- 입력 탭에서 무엇을 입력해야 하는지와 시뮬레이션/계산 상세에서 무엇을 보여줘야 하는지 역할을 더 선명하게 정리했다.
- `없음 확정`, `호주상속 가이드`, `차순위 여성 상속인 동일가적 검토`가 어떤 사건에서 어떻게 열리는지 한 문서에서 볼 수 있게 만들었다.

## 추가 문서

- `!명세서/specs/SUBSTITUTION_AND_REINHERITANCE_CASE_MATRIX_2026-04-14_UTF8.md`

## 핵심 포인트

- 대습상속 = 선사망자 기준
- 재상속 = 후사망자 기준
- 후속 상속인이 있으면 입력
- 없으면 없음 확정
- 구법에서 결과가 달라질 수 있으면 호주상속/재산상속 판단 가이드
- 차순위 상속으로 넘어갈 때는 여성 상속인의 동일가적 여부를 필요한 경우에만 재검토
