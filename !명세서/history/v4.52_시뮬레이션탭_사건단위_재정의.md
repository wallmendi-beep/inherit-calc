# v4.52 시뮬레이션 탭 사건 단위 재정의

작성일: 2026-04-13

## 변경 요약
- `시뮬레이션` 탭을 `calcSteps 카드 나열` 구조에서 `사건 선택 + 사건별 1차 흐름 보기` 구조로 재정의했다.
- 재상속은 현재 사건 안에서 계속 전개하지 않고 `별도 사건`으로 끊었다.
- 대습상속 가지도 현재 사건의 세부 계산표로 끝없이 늘리지 않고 `관련 사건 보기`로 넘기도록 했다.

## 이유
- 기존 흐름 보기는 `계산 상세`를 카드형으로 다시 보여주는 느낌이 강했다.
- 사용자가 원한 것은 `김혁조의 상속 흐름도`, `구수명의 상속 흐름도`처럼 사건별 시뮬레이션이었다.
- 같은 사람이라도 사망 시점과 적용 법이 달라 지분 관계가 달라지는 이유를 사건 단위로 보여줄 필요가 있었다.

## 구현 방향
- 좌측: 사건 목록
- 우측: 선택 사건의 헤더 + 1차 상속 흐름 카드
- 이름 클릭 시 입력 탭으로 이동
- 관련 사건이 있는 경우 `관련 사건 보기` 버튼 제공
