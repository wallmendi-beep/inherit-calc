# v4.48 스마트가이드 호주안내 예외처리

## 배경
- 윤종옥처럼 `사위(대습불가)`로 지분이 0인 사람에게도 스마트 가이드가 `호주 여부 확인` 문구를 띄우는 문제가 있었다.

## 원인
- 스마트 가이드의 `missing-hoju`, `chained-hoju` 생성 규칙이
  - `blocked_husband_substitution`
  같은 법리상 배제 상태를 고려하지 않았다.

## 변경
- hard exclusion 상태(`lost`, `disqualified`, `remarried`, `renounce`, `blocked_husband_substitution`)인 경우
  - 호주 관련 스마트 가이드를 생성하지 않도록 수정

## 기대 결과
- 실제 지분이 0이고 호주가산 검토 대상이 아닌 사람에게
  - 불필요한 호주 확인 가이드가 뜨지 않음
