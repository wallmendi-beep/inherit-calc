# 업데이트 노트 (v4.84) - 정문자 후속상속 네비게이션 및 선사망 뱃지 정리

작성일: 2026-05-03

## 수정 배경

정문자 관련 가이드가 다음처럼 중복 또는 오동작했다.

- `후속 상속 미확정 — [김명수] 사건: [정문자]`와 `혼인 정보 확인 — [정문자]`가 동시에 표시됨
- 정문자 후속상속 카드를 눌렀을 때 김혁조 사건 쪽으로 돌아가 후속상속인 없음 버튼을 누르기 어려움
- 입력 행의 `선사망` 뱃지가 호주상속 뱃지와 같은 파란 계열이라 시각적으로 헷갈림

## 반영 내용

1. 후속상속 미확정이 있는 동안 혼인 정보 권장 카드 숨김
   - 같은 인물에게 직접 후속상속 미확정 카드가 있으면 혼인 정보 확인 카드는 표시하지 않는다.
   - 후속상속인 입력 또는 없음 확정이 먼저 처리되어야 하므로 가이드 우선순위를 정리했다.

2. 후속상속 카드 네비게이션 보정
   - `personId`만으로 이동하면 중복 등장 인물 중 먼저 발견되는 위치로 갈 수 있었다.
   - 단일 후속상속 미확정 카드에는 실제 `node.id`를 `targetNodeId`로 포함해 정확한 노드로 이동하게 했다.
   - 입력 탭에서는 해당 노드의 후속상속 입력 탭으로 이동해 `없음 확정` 버튼을 누를 수 있게 했다.

3. 선사망 뱃지 색상 변경
   - 파란 계열에서 rose 계열로 변경
   - 호주상속/재상속 계열과 시각적으로 분리

## 기대 결과

- 정문자는 후속상속 미확정 카드 하나만 우선 표시된다.
- 정문자 카드를 누르면 정문자 후속상속 검토 위치로 이동한다.
- 선사망 뱃지는 호주상속 뱃지와 구분된다.
