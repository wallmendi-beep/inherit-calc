# 업데이트 노트 (v4.83) - 스마트가이드 중복 권장 카드 및 닫기 버튼 정리

작성일: 2026-05-03

## 수정 배경

스마트가이드에서 같은 사람에 대해 우선 확인 카드와 권장 카드가 동시에 표시되는 문제가 있었다.

대표 사례:
- `후속 상속 미확정 — [정문자]`가 우선 확인 항목과 권장 가이드에 중복 표시
- `후속 상속 미확정 — [이경숙]`도 같은 방식으로 중복 표시
- 정문자에 대해 후속상속 미확정 카드가 이미 있는데도 차순위 자동 분배 안내가 권장 가이드로 다시 표시

또한 권장 가이드에는 `확인` 버튼과 `X` 숨김 버튼이 함께 있어, 사용자가 보기에는 두 닫기 동작이 중복되어 보였다.

## 반영 내용

1. 중복 권장 카드 억제
   - 후속상속 미확정 우선 카드가 이미 생성된 사람은 import 기반 `missing-descendants` 권장 카드를 만들지 않는다.
   - 같은 사람에게 후속상속 미확정 우선 카드가 있으면 `auto-sibling-redistribution` 권장 안내도 숨긴다.

2. 권장 가이드 닫기 동작 정리
   - 권장 가이드의 별도 `X` 버튼을 제거
   - `확인` 버튼만 남겨 확인 완료 목록으로 이동하는 흐름으로 통일

## 기대 결과

- 우선 확인 항목은 실제 입력이 필요한 항목 중심으로 유지된다.
- 권장 가이드는 우선 항목과 같은 내용을 반복하지 않는다.
- 사용자는 `확인` 하나로 검토 완료 처리를 이해할 수 있다.
