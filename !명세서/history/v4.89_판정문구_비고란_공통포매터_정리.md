# 업데이트 노트 (v4.89) - 판정 문구 비고란 공통 포매터 정리

작성일: 2026-05-07

## 1. 배경

사건검토 탭의 상속흐름표, 인쇄 보고서, 계산 상세표에서 같은 계산 결과가 서로 다른 표현으로 표시되는 문제가 있었다.

대표 사례:

- `출가녀 감산 (남자의 1/4)`
- `비동일가적`
- `여자 감산 (남자의 1/2)`
- `처 감산 (직계비속의 1/2)`
- `1991년 이전 처 사망으로 사위 대습상속권 없음`

비고란은 길게 설명하는 곳이 아니라 판정 요소를 빠르게 검토하는 곳이므로, 짧고 일관된 문구가 필요하다.

## 2. 반영 기준

### 2.1 동일가적·비동일가적

법률상 기준은 단순한 출가 여부가 아니라 `동일가적/비동일가적`이다.

따라서 비고란은 다음처럼 표시한다.

| 상황 | 표시 |
|---|---|
| 1960년 구민법상 동일가적 여성 감산 | `동일가적 여자, 감산(남자의 1/2)` |
| 구법상 비동일가적 여성 감산 | `비동일가적, 감산(남자의 1/4)` |
| 혼인일이 확인되는 비동일가적 여성 감산 | `YYYY년 MM월 DD일 혼인, 비동일가적, 감산(남자의 1/4)` |
| 일반 남자·남편·현행민법 균분자 | `균분` |

남자·남편에게 `동일가적`을 단독 표시하지 않는다.

### 2.2 처 감산

1960년 구민법상 처가 직계비속과 공동상속하는 경우의 기준 문구를 다음처럼 정정했다.

```text
처 감산(남자의 1/2)
```

기존 `처 감산(직계비속의 1/2)`은 계산값은 같을 수 있으나 법률상 기준 문구로는 부정확하므로 사용하지 않는다.

### 2.3 상속권 없음

표 비고란에서는 장문 설명 대신 짧은 사유를 사용한다.

| 기존 원문 | 표시 |
|---|---|
| 1991년 이전 처 사망으로 사위 대습상속권 없음 | `1991년 이전 처 사망(대습상속 불가)` |
| 1991년 이전 피상속인 사망으로 남편 대습권 없음 | `1991년 이전 상속개시(남편 대습 불가)` |
| 대습상속 개시 전 재혼으로 인한 상속권 소멸 | `재혼(대습상속 불가)` |
| 선순위(직계비속) 상속인이 존재하여 상속권 없음 | `선순위 직계비속 존재` |

## 4. 상속흐름표 사건번호와 흐름 표기

인쇄용 산출표는 실제 계산 단계인 `calcSteps`를 기준으로 사건번호를 붙인다.

화면용 상속흐름표는 이해를 돕기 위해 하나의 사건을 취득 원천별로 분해해 보여줄 수 있다. 이때 화면용 분해 노드를 새 사건처럼 번호 매기지 않는다.

표시 기준:

| 상황 | 표시 |
|---|---|
| 유입이 하나인 사건 | `제N사건` |
| 유입이 여러 개인 사건 | `제N사건 · 흐름 M` |
| 흐름 설명 | `상위 피상속인 → 현재 피상속인 · 지분` |

예:

```text
제3사건 · 흐름 1
흐름: 김혁조 → 김명남 · 1/15

제3사건 · 흐름 2
흐름: 구수명 → 김명남 · 2/165
```

따라서 화면에서 표시되는 흐름 카드 수가 인쇄용 사건 수보다 많더라도, 사건번호 자체는 사망사실 기준의 실제 사건번호를 유지한다.

## 5. 회귀 검증 포인트

김혁조 사건 등 구법 사건에서 다음처럼 보이는지 확인한다.

```text
윤종옥 / 남편 / 균분
구수명 / 처 / 처 감산(남자의 1/2)
윤숙자 / 딸 / 비동일가적, 감산(남자의 1/4)
정인숙 / 딸 / 동일가적 여자, 감산(남자의 1/2)
조창제 / 남편 / 1991년 이전 처 사망(대습상속 불가)
```

계산 분수는 기존 엔진 결과를 유지하고, 표시 문구만 공통 기준으로 정리한다.
