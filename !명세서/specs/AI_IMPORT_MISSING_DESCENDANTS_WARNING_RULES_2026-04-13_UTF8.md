# AI 불러오기 하위상속인 누락 경고 규칙

## 목적
- 이미지 기반 AI 입력에서 자주 발생하는 `사망자 하위상속인 누락`을 사용자가 즉시 눈치챌 수 있게 한다.
- 계산기 엔진이 자동분배로 넘어가기 전에, 입력 데이터 누락 가능성을 먼저 경고한다.

## 배경
- 실제 서류상 후속 상속인이 존재하는데도 AI가 JSON에 `heirs: []`로 남기는 경우가 있다.
- 이 경우 사용자는 `불러오기`가 안 되거나, 자동분배 안내만 보고 실제 누락을 놓칠 수 있다.
- 예: 정문자처럼 계산 자료에는 후속 상속 흔적이 있으나 원본 입력 JSON에는 하위상속인이 비어 있는 경우

## SSOT 규칙
1. 사망한 인물의 `heirs`가 비어 있으면, 단순히 자동분배 안내만 하지 말고 먼저 **하위상속인 누락 확인 경고**를 띄운다.
2. 선사망 자녀라면:
   - `대습상속인(배우자 또는 직계비속) 누락 여부를 확인해 주세요.`
3. 후사망 배우자 또는 기타 사망자라면:
   - `하위상속인이 없습니다. 확인해 주세요.`
   - 그 다음 줄에 자동분배 결과를 보조 설명으로 붙일 수 있다.
4. 선사망 배우자는 상속권 없음이므로 이 경고를 생략할 수 있다.

## 구현 기준
- `src/hooks/useSmartGuide.js`
  - `node.isDeceased && activeHeirs.length === 0` 분기에서
  - 기존 `후속 상속인이 비어 있습니다` 문구를
    `하위상속인이 없습니다. 확인해 주세요` 중심으로 교체

## 기대 효과
- AI 불러오기 누락을 사용자가 더 빨리 발견할 수 있다.
- 자동분배 안내가 “정답”처럼 보이는 오해를 줄인다.
