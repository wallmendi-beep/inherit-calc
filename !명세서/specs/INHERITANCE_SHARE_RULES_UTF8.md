# 상속지분 계산 규칙

본 문서는 상속지분 계산기의 시대별 분배 규칙, 배우자 처리, 대습상속, 호주상속 가산 기준을 요약한 계산 기준 문서다.

## 1. 최상위 원칙

- 상속지분은 항상 `해당 상속이 개시된 시점의 법`으로 계산한다.
- 한 사건 안에서도 `루트 상속`, `재상속`, `대습상속`은 서로 다른 시점 법을 적용할 수 있다.
- 대습상속인은 피대습자의 권리를 기계적으로 승계하는 것이 아니라, 법률상 인정된 범위에서 직접 상속받는 구조로 본다.

## 2. 시대 구분

### 2.1. 1960-01-01 ~ 1978-12-31

- 균분상속을 기본으로 함
- 호주상속 동시 발생 시 5할 가산
- 여성 상속분 차등
- 출가녀 차등
- 처는 직계비속과 공동상속 시 남자 상속분의 1/2

### 2.2. 1979-01-01 ~ 1990-12-31

- 균분상속 기본
- 호주상속 동시 발생 시 5할 가산
- 처는 5할 가산
- 출가녀 1/4 차등 유지

### 2.3. 1991-01-01 이후

- 배우자 1.5 가산
- 자녀 간 성별 차등 없음
- 출가녀 차등 없음

## 3. 배우자 규칙

### 3.1. 생존 배우자

- 해당 시기 법에 따라 배우자 가산 또는 감산을 계산한다.

### 3.2. 선사망 배우자

- 선사망 배우자는 해당 상속에서 상속권이 없다.
- 선사망 배우자는 원칙적으로 피대습자가 아니다.
- 따라서 선사망 배우자를 기준으로 자동 대습상속을 열면 안 된다.

### 3.3. 이혼·재혼

- 기준 사망일 이전 이혼은 배우자 상속권 탈락
- 기준 사망일 이전 재혼은 관련 시기 법리에 따라 배우자 상속권 탈락 또는 대습배우자 탈락

## 4. 대습상속 규칙

다음이 원칙적 대습상속 유발 사유다.

- 직계비속 선사망
- 형제자매 선사망
- 법문상 허용되는 결격 또는 상실

다음은 원칙적으로 대습상속 유발 사유가 아니다.

- 배우자 선사망
- 단순 상속포기
- 하위 상속인이 없는 선사망

## 5. 호주상속 가산 규칙

### 5.1. 일반 호주가산

- 본인 상속 단계에서
- 해당 상속인이 실제 호주를 승계하고
- 적용 시기 법이 호주가산을 허용할 때만 검토

### 5.2. 대습 호주가산

- 피대습자 자격이 실제 성립하고
- 대습상속인 중 호주승계인이 있고
- 판례·선례형 확장 해석을 허용하는 설정 또는 규칙을 선택한 경우에만 검토

### 5.3. 금지

- 선사망 배우자 계통에 대한 호주가산
- 단순히 `isHoju = true`라는 이유만으로 자동 가산

## 6. 형제자매 상속과 구법 차등

구법 시기 형제자매 상속에서도 시대별 차등 규칙이 살아 있을 수 있다.

- 남자 형제와 여자 형제 차등
- 동일가적 여부
- 출가녀 감산

따라서 형제자매 상속은 무조건 균분으로 처리하지 않는다.

## 7. 구현 지침

- `distributionDate` 기준으로 시대를 판정한다.
- `isSubstitutionTrigger()`와 `canApplyHojuBonus()`는 별도 규칙으로 유지한다.
- 배우자 선사망은 `탈락` 처리 후 종료하고, 대습상속 가지로 넘기지 않는다.
- 테스트는 최소한 아래 케이스를 포함해야 한다.
  - 생존 배우자 공동상속
  - 선사망 자녀 대습상속
  - 선사망 배우자 탈락
  - 피대습자 호주 여부에 따른 대습 호주가산 허용/배제
