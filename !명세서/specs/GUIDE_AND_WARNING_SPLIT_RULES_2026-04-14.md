# 가이드와 경고등 분리 규칙

작성일: 2026-04-14

## 목적
- 데이터 입력 탭은 사실 입력 중심 안내만 제공한다.
- 시뮬레이션, 계산 상세, 결과 탭은 계산 결과와 무결성 경고를 중심으로 보여준다.
- 같은 문제를 입력 가이드와 결과 경고에서 중복으로 반복하지 않는다.

## 데이터 입력 탭 가이드에 남기는 항목
- 사건번호/피상속인 기본정보 누락
- 사망일 누락
- 배우자 중복 입력
- 선사망 상속인 중 대습상속인 입력 필요
- 직접 입력된 후속 상속인 필요 여부
- 호주상속 판단을 위한 상속인 불러오기 필요
- AI/파일 불러오기 직후 입력값 확인 필요

## 데이터 입력 탭 가이드에서 제거하는 항목
- 계산 엔진 무결성 경고
- 최종 지분 합계 불일치
- 사망자 지분이 최종 결과에 남아 있음
- 중간 단계 지분 미전달
- 사건별 상속권 인정/배제 결론

## 시뮬레이션/계산 상세/결과 탭에서 다루는 항목
- 최종 분배 지분 합계 불일치
- 중간 단계 지분 미전달
- 사망자 지분의 최종 결과 잔존
- 순환참조, 계보 위반, 법리 적용 결과
- 사건 한정 상속권 배제/가산/감산

## 경고등 원칙
- 입력 탭: 사실 입력이 필요한 항목 중심으로만 점등
- 시뮬레이션/계산 상세/결과 탭: 계산 결과 검산 경고를 점등
- 동일 문제를 입력 가이드와 결과 경고에 동시에 크게 반복하지 않는다
