# 상속흐름표 판정요소 표시 기준

작성일: 2026-05-07  
관련 화면: 사건검토 탭 > 상속흐름표  
관련 코드: `src/utils/judgmentLabels.js`, `src/components/TreePanel.jsx`, `src/components/PrintReport.jsx`, `src/components/CalcPanel.jsx`, `src/engine/distribution.js`

## 1. 목적

상속흐름표의 `판정 요소` 칸은 계산 결과의 법률상 이유를 사용자가 빠르게 검토하기 위한 비고란 성격의 영역이다.

따라서 다음 원칙을 따른다.

1. 같은 법적 효과는 같은 표현으로 표시한다.
2. `동일가적`, `비동일가적`은 구민법상 법률효과가 있는 경우에 판정요소로 표시한다.
3. 가감산 또는 상속권 없음처럼 지분에 영향을 준 요소만 판정요소로 표시한다.
4. 날짜가 지분 판정에 영향을 준 경우에는 가능한 한 날짜를 함께 표시한다.
5. `상속권 없음`은 결과이고, `판정 요소`에는 그 이유를 짧은 비고 문구로 표시한다.
6. 비고란 문구는 서술형 문장보다 쉼표로 구분한 짧은 표기를 우선한다.

## 2. 현재 확인된 표현 문제

상속흐름표에서 다음과 같은 표현 불일치가 확인되었다.

| 현재 표현 | 문제 |
|---|---|
| `비동일가적` | 감산 효과가 있으면 `비동일가적, 감산(남자의 1/4)`처럼 효과까지 함께 표시해야 한다. |
| `동일가적` | 남자·남편·일반 균분 상속인에게는 판정요소로 표시할 필요가 없다. 다만 1960년 동일가적 여성 감산에는 표시한다. |
| `출가녀 감산 (남자의 1/4)` | 법률상 기준은 출가 여부 자체보다 비동일가적 여부이므로 `비동일가적, 감산(남자의 1/4)`로 통일한다. |
| `여자 감산 (남자의 1/2)` | 1960년 제정 민법상 동일가적 여성 감산은 `동일가적 여자, 감산(남자의 1/2)`로 표시한다. |
| `1991년 이전 처 사망으로 사위 대습상속권 없음` | 원인과 결과가 길게 섞여 있어 `1991년 이전 처 사망(대습상속 불가)`처럼 정리하는 편이 낫다. |

## 3. 법 시기별 판정요소 표시 기준

### 3.1 구민법(1960년 제정 민법)

적용 기간: 1960-01-01부터 1978-12-31까지 사망한 사건

| 대상 | 계산 효과 | 판정요소 표시 |
|---|---:|---|
| 일반 남자 | 1 | `균분` |
| 남편 | 1 | `균분` |
| 동일가적 여성 | 1/2 | `동일가적 여자, 감산(남자의 1/2)` |
| 비동일가적 여성 | 1/4 | `비동일가적, 감산(남자의 1/4)` |
| 처가 직계비속과 공동상속 | 1/2 | `처 감산(남자의 1/2)` |
| 호주상속인 | 1.5 | `호주상속 5할 가산` |

표시 원칙:

- 남자와 남편은 `동일가적`이라고 표시하지 않는다.
- `동일가적/비동일가적`은 여성 감산처럼 실제 지분 효과가 있는 경우에만 표시한다.
- 혼인으로 비동일가적이 된 경우에는 `YYYY년 MM월 DD일 혼인, 비동일가적, 감산(남자의 1/4)`로 표시한다.
- 혼인일 없이 비동일가적 상태만 확인된 경우에는 `비동일가적, 감산(남자의 1/4)`로 표시한다.

예:

```text
윤종옥 [남편] → 균분
윤우영 [아들] → 균분
윤정희 [딸, 혼인일 있음] → 1920년 03월 01일 혼인, 비동일가적, 감산(남자의 1/4)
정인숙 [딸, 동일가적] → 동일가적 여자, 감산(남자의 1/2)
```

### 3.2 1979년 개정 민법

적용 기간: 1979-01-01부터 1990-12-31까지 사망한 사건

| 대상 | 계산 효과 | 판정요소 표시 |
|---|---:|---|
| 일반 남자 | 1 | `균분` |
| 남편 | 1 | `균분` |
| 동일가적 여성 | 1 | `균분` |
| 비동일가적 여성 | 1/4 | `비동일가적, 감산(남자의 1/4)` |
| 처 | 1.5 | `처(배우자) 5할 가산` |
| 호주상속인 | 1.5 | `호주상속 5할 가산` |

표시 원칙:

- 1979년 개정 민법에서는 동일가적 여성도 일반 균분이면 `균분`으로 표시한다.
- `비동일가적` 단독 표시는 사용하지 않는다. 실제 지분 효과가 있으면 `비동일가적, 감산(남자의 1/4)`로 표시한다.

### 3.3 현행 민법(1991년 이후)

적용 기간: 1991-01-01 이후 사망한 사건

| 대상 | 계산 효과 | 판정요소 표시 |
|---|---:|---|
| 자녀 | 1 | `균분` |
| 배우자 | 1.5 | `배우자 5할 가산` 또는 `처(배우자) 5할 가산`, `남편(배우자) 5할 가산` |

표시 원칙:

- 현행 민법 사건에서는 `동일가적`, `비동일가적`, `비동일가적 감산`, `여자 감산`을 표시하지 않는다.
- 성별·가적 상태가 지분에 영향을 주지 않으면 `균분`으로 통일한다.

## 4. 대습상속·재상속 흐름 표현

`흐름` 칸은 결과 흐름만 표시한다.

| 상황 | 흐름 표시 |
|---|---|
| 후사망자가 자기 지분을 받은 뒤 다시 사망 | `재상속 제N사건` |
| 선사망자 계통으로 지분이 이어짐 | `대습상속 제N사건` |
| 최종 생존 상속인에게 귀속 | `최종 취득` |
| 상속권 없음 | `상속권 없음` |

`판정 요소` 칸에는 흐름이 아니라 지분이 바뀐 이유 또는 상속권 없음 이유를 적는다.

## 5. 상속권 없음 표현 기준

상속권 없음은 `흐름` 칸에 `상속권 없음`으로 표시하고, `판정 요소` 칸에는 이유를 간결하게 쓴다.

| 현재 사유 | 비고란 권장 표현 |
|---|---|
| 1991년 이전 처 사망으로 사위 대습상속권 없음 | `1991년 이전 처 사망(대습상속 불가)` |
| 피상속인보다 먼저 사망, 후속 없음 | `YYYY년 MM월 DD일 선사망(대습상속인 없음)` |
| 재혼으로 상속권 없음 | `YYYY년 MM월 DD일 재혼(상속권 없음)` |
| 이혼으로 상속권 없음 | `YYYY년 MM월 DD일 이혼(상속권 없음)` |
| 상속포기 | `상속포기` |
| 상속결격 | `상속결격` |
| 상속권 상실 | `상속권 상실` |

예:

```text
조창제 [남편] / 흐름: 상속권 없음 / 판정 요소: 1991년 이전 처 사망(대습상속 불가)
```

## 6. 날짜 표시 기준

날짜가 판정 근거가 되는 경우에는 날짜를 표시한다.

| 판정 | 날짜 기준 | 표시 예 |
|---|---|---|
| 비동일가적 감산 | 혼인일 또는 비동일가적 전환일 | `1925년 02월 03일 혼인, 비동일가적, 감산(남자의 1/4)` |
| 복적으로 동일가적 회복 | 복적일 | `1970년 05월 01일 복적, 균분` |
| 선사망 배제 | 사망일 | `1940년 04월 22일 선사망(대습상속인 없음)` |
| 재혼 배제 | 재혼일 | `1980년 01월 01일 재혼(상속권 없음)` |
| 이혼 배제 | 이혼일 | `1980년 01월 01일 이혼(상속권 없음)` |

현재 데이터 구조에서는 `marriageDate`를 혼인으로 인한 비동일가적 전환 판단일로 사용한다.

## 7. UI 표시 규칙

### 7.1 `판정 요소` 칸 우선순위

1. 상속권 없음 사유
2. 호주상속 가산
3. 배우자 가산·감산
4. 비동일가적 감산
5. 여자 감산
6. 균분

### 7.2 사용하지 않을 표현

아래 표현은 단독 판정요소로 사용하지 않는다.

- 남자·남편·현행민법 균분자에게 붙는 `동일가적`
- 감산 효과 없이 단독으로만 표시되는 `비동일가적`
- `상속권 없음`만 단독 반복 표시

다만 입력 탭의 상태 토글이나 검수 가이드에서는 `동일가적/비동일가적` 용어를 계속 사용할 수 있다.

## 8. 코드 반영 위치

### 8.1 `judgmentLabels.js`

화면·인쇄용 비고란 문구는 공통 포매터에서 정리한다.

- `formatModifierLabel()`
- `formatExclusionLabel()`
- `formatJudgmentLabel()`
- `formatJudgmentSummary()`
- `formatRegisterNote()`

### 8.2 `distribution.js`

엔진은 계산 효과를 만든다.

- `modifierReason`
- `exclusionReason`

엔진 문구는 법적 효과를 보존하되, UI 문장 전체를 책임지지 않는다.

### 8.3 표시 컴포넌트

상속흐름표, 사건 보고서, 인쇄 보고서, 계산 상세표는 공통 포매터를 통해 표시 문구를 정리한다.

수정 대상:

- `src/components/TreePanel.jsx`
- `src/components/PrintReport.jsx`
- `src/components/CalcPanel.jsx`

## 9. 김혁조 사건 검증 포인트

다음 표현은 회귀 검증 기준으로 삼는다.

```text
윤종옥 / 남편 / 4/225 / 재상속 제N사건 / 균분
김경혜 / 자녀 / 4/75 / 최종 취득 / 균분
윤숙자 / 딸 / 4/2925 / 대습상속 제N사건 / 비동일가적, 감산(남자의 1/4)
정인숙 / 딸 / 2/315 / 최종 취득 / 동일가적 여자, 감산(남자의 1/2)
조창제 / 남편 / - / 상속권 없음 / 1991년 이전 처 사망(대습상속 불가)
```

혼인일 등 비동일가적 전환 근거일이 있는 사람은 가능한 경우 다음처럼 날짜를 포함한다.

```text
1925년 02월 03일 혼인, 비동일가적, 감산(남자의 1/4)
```
