## [베타 v1.1.0] — 2026-04-29 (개발 중 · PR #1~4 머지 예정)

### ✅ 버그 수정
- **CSV 내보내기 빈 파일 문제 수정**: 지분 요약 내보내기(CSV)가 항상 빈 파일로 저장되던 버그 수정 — `finalShares.survivors` 잘못된 필드 참조를 `direct + subGroups` 병합으로 교체
- **`result` 탭 참조 오류 수정**: 삭제된 `result` 탭을 `PrintReport`, `App.jsx` 등 여러 곳에서 여전히 참조하던 dead code 제거
- **`useSmartGuide.js` 한글 인코딩 깨짐 수정**: 코드 내 `'이름 미상'` 등 문자열이 깨진 채로 저장되어 UI에 표시되던 문제 수정
- **사망일 유효성 검증 추가**: 존재하지 않는 날짜(예: `2024-02-30`, 미완성 입력) 입력 시 빨간 테두리와 "올바른 날짜가 아닙니다" 툴팁 표시, 저장 차단
- **순환 참조 실시간 감지**: 상속인을 자기 자신의 하위에 연결하는 등 순환 구조 발생 시 편집 중 즉시 필수 가이드로 표시 (기존: 계산 시점에만 감지)

### 🆕 신규 기능
- **배우자별 자녀 중복 인물 감지**: 동일인이 wife/husband 탭에 이중 입력된 경우 `duplicate-person` 경고 가이드 표시 — '확인 (다른 사람)' 버튼으로 완료 처리 가능
- **성명 중복 검사**: 같은 부모 아래 동일 이름 상속인이 2명 이상이면 경고 가이드 — '확인 (다른 사람)' 버튼으로 완료 처리 가능
- **재혼 배우자 자녀 범위 확인 가이드**: 재혼 이력(`remarriageDate`)이 있는 배우자 아래 자녀가 있을 때 자녀 범위 확인 권장 가이드 표시
- **입력 탭 blocking 이슈 배너**: 현재 사건 탭에 계산을 막는 이슈가 있으면 입력 탭 상단에 황색 경고 배너 표시

### 🎨 UX 개선
- **스마트 가이드 패널 개선**:
  - 필수 항목 완료율 진행 바 추가 (X/N, 완료 시 초록색으로 전환)
  - 필수/권장 섹션 접기/펼치기 토글
  - 섹션 제목에 건수 배지 표시
  - 필수 항목 전부 완료 시 "필수 항목 완료" 초록 배너 표시
- **사건 그래프 카드 단순화**:
  - 카드에 이름 + 결과 지분 + 재상속/대습 버튼만 표시 (배지·계산식·날짜 제거)
  - 카드 크기 대폭 축소 (118px → 68px)
  - 연결선 레이블 제거 (중복 정보)
  - 그래프 하단 "계산 근거" 서술 블록 신규 추가: 취득자 이유·날짜·대습/재상속 연결, 상속권 없음 취소선+이유 표시
- **선사망 힌트**: 상속인 사망일 입력 시 피상속인 사망일보다 이를 경우 파란 테두리 + "선사망 처리" 힌트 툴팁 표시
- **관계 무효화 감지**: 부모 성별이 바뀌어 현재 관계가 유효하지 않을 때 빨간 텍스트 + "관계 재검토 필요" 툴팁

### 🔧 다크 모드 개선
- 보조 텍스트 명도 전면 상향 (`neutral-400/500` → `neutral-300/400`) — 97개 인스턴스

- SVG 그래프 엣지 다크모드 대응
- 앱 전역 배경 따뜻한 톤으로 조정

- **App.jsx 리팩토링**: 982줄 → 835줄, 4개 커스텀 훅 분리 (`useVaultState`, `useDeceasedTabs`, `useCalcResult`, `useAmountCalc`)
- **미사용 컴포넌트 정리**: `CalcPanelV2`, `TopToolbarFixed`, `ResultPanel` 등 중복·미사용 파일 제거

- **버전 태그 주석 제거**: 코드 내 `[v4.xx]` 형태 주석 전체 제거

## [베타 v1.0.1] — 2026-04-24

### ✅ 버그 수정
- **AI 안내문 인쇄 다이얼로그 즉시 닫힘 버그 수정**: AI 입력 모달의 프린터 버튼을 누르면 인쇄 다이얼로그가 열리자마자 바로 닫히던 문제를 `afterprint` 이벤트 처리로 해결했습니다.

### 🎨 디자인 개선
- **상단 뱃지 제거**: 앱 타이틀 옆의 버전 뱃지를 제거했습니다. 버전 정보는 창 제목에서 확인할 수 있습니다.
- **사건 그래프 슬라이드 다이얼 경량화**: 좌측 사건 목록 스크롤 다이얼(DialScrubber)의 트랙 두께와 점 크기를 줄여 시각적으로 더 가볍고 세련된 느낌으로 개선했습니다.
- **입력 화면 하단 워터마크 추가**: 데이터 입력 화면 우측 하단에 필기체 `Designed by J.H. LEE` 음각 서명을 추가했습니다.

### 📝 UX 문구 개선
- **스마트 가이드 정상 상태 문구 개선**: "정상 / 입력 기준으로…" → "이상 없음 / 가계도 검증이 완료되었습니다. / 사건 검토 및 계산 결과를 확인해 주세요." 로 더 직관적으로 변경했습니다.
- **결과표 취득 경로 텍스트 간소화**: "망 OOO의 아들으로 1979 개정민법 적용" → "망 OOO의 아들 `<1979년 개정민법>`" 형식으로 축약하여 가독성을 높였습니다.
- **사건 검토 탭 이름 통일**: 스마트 가이드 안내 문구의 탭 이름을 실제 탭 이름과 일치시켰습니다.

### 📚 문서화
- **사용자 매뉴얼** (`USER_MANUAL.md`) 신규 작성: 화면 구성, 메뉴바, 패널, 탭별 기능 상세 설명.
- **README.md** 전면 개편: 개인정보 보호 방침, 연도별 상속권 판정 규칙, 스마트 가이드 활용법 등 필수 사항 정리.

## [베타 v1.0.0] — 2026-04-22

### 🎉 최초 베타 릴리즈

- **4단계 작업 탭 구성**: 데이터 입력 → 사건 검토(시뮬레이션 그래프) → 계산 상세 → 결과 요약
- **연도별 민법 자동 적용**: 피상속인 사망일 기준으로 1960 / 1979 / 1991년 민법을 자동 판별하여 지분 계산
- **스마트 가이드**: 입력 데이터의 논리 오류 및 법률 쟁점을 실시간으로 감지하고 안내
- **사건 그래프 뷰**: 재상속·대습상속 흐름을 직각 연결선 그래프로 시각화
- **AI 자동 입력 모달**: 자연어 텍스트 → AI → JSON 응답을 붙여넣으면 자동으로 가계도 구성
- **호주상속 처리**: 1991년 이전 사망 사건의 호주 자격 부여 및 호주가산(5할) 지분 계산 지원
- **대습상속 자동 판정**: 피상속인보다 먼저 사망한 상속인의 배우자·자녀에게 대습상속 자동 적용
- **오프라인 단독 HTML 배포**: 인터넷 없이 단일 HTML 파일만으로 완전한 기능 사용 가능
- **Windows 설치파일(NSIS) 배포**: 데스크탑 앱으로 설치 가능
- **다크모드 지원**: 라이트/다크 모드 즉시 전환
- **인쇄/PDF 출력 최적화**: 상세 계산서·요약표를 깔끔한 인쇄 레이아웃으로 출력 가능
- **저장/불러오기**: `.json` 파일로 작업 내용 저장 및 재사용
