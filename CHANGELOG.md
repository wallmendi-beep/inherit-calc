## [베타 v1.1.0] — 2026-04-29 (개발 중 · PR #1~4 머지 예정)

### 🧮 계산 엔진 보정 (2026-05-06)
- **동명이인 실무 표기 반영**: AI 입력 및 검수 기준에서 동일 성명 인물이 실제 다른 사람으로 확인되면 `김경희(A)`, `김경희(B)`처럼 이름 자체를 구분하도록 정리했습니다. 같은 사람의 반복 등장은 같은 `personId`와 같은 구분 이름을 유지합니다.
- **반복 등장 동일인 처리 보정**: 동일 `personId`가 여러 상속사건에 반복 등장하더라도 이를 `duplicate` 상속 제외로 보지 않고, 하위 전개만 중단해 재상속·대습 흐름의 중복 순회를 막도록 정리했습니다.
- **선사망·후속없음 분배대상 제외 기준 명확화**: 현재 사건 기준일보다 먼저 사망했고 유효한 대습/후속 상속인이 없는 인물은 분배분모에 포함하지 않도록 계산대상 필터 기준을 보강했습니다.
- **김혁조 사건 회귀 기준 반영**: 김혁조 최신 JSON 검산에서 제1사건 분배(`구수명 2/15`, `김명남·김명수·김말수 각 1/15`, `김일상 2/5`, `김일주 4/15`)와 선사망자 제외 기준을 회귀 검증 포인트로 기록했습니다.

### 🧾 검토 화면·요약표 개선 (2026-05-06)
- **그룹통분 지분 추가**: 상속지분 요약표에 `그룹통분 지분` 컬럼을 추가해 윤우영 재상속 그룹처럼 같은 사건 그룹 안에서는 동일 분모로 지분을 비교할 수 있게 했습니다.
- **좌우 패널 스크롤 연동**: 사건 검토의 상속흐름표에서 좌측 사건 카드를 클릭하면 우측 사건 표가 중앙으로 이동하고, 우측 선택 상태가 바뀌면 좌측 카드도 자동으로 스크롤되어 강조되도록 개선했습니다.
- **좌측 패널 독립 스크롤**: 사건 보고서 패널과 상속흐름표 좌측 목록에 독립 스크롤을 적용해 사건 수가 많아도 패널 내부에서 탐색할 수 있게 했습니다.

### 🧾 상속지분 탭 재정리 (2026-05-04)
- **취득경로 탭 흡수**: 별도 `취득경로` 탭을 제거하고, `상속지분` 탭 내부의 `취득합산 / 최종지분표` 보기로 통합했습니다.
- **좌측 취득계보 패널**: 사건 검토 탭의 좌측 보고서 패널과 같은 화면 구조로, 선택 상속인의 취득계보를 좌측에 고정 표시합니다.
- **취득합산표 단순화**: 우측 표는 `상속인 / 경로합산 / 최종지분 / 통분지분`만 표시해 숫자 검산 역할에 집중하도록 정리했습니다.
- **가이드 하이라이트 완화**: 가이드 클릭 시 강한 파란 테두리와 확대 효과를 제거하고, 연한 배경과 얇은 회청색 테두리로 자연스럽게 안내합니다.

### 🧭 사건 검토 탭 재설계 (2026-05-04)
- **상속지분 마인드맵 도입**: 기존 사건 그래프/관계 트리 중심 검토를 상속지분 분기 마인드맵으로 통합했습니다.
- **좌측 사건 보고서 사이드패널**: 사건 검토 탭에서는 좌측 패널이 가계도 요약이 아니라 선택 노드가 속한 사건의 보고서로 작동합니다.
- **노드 카드 경량화**: 노드는 `관계 / 이름 / 지분 / 분기정보` 중심으로 줄이고, 사망일·제외 사유·가감산 상세는 사건 보고서로 이동했습니다.
- **사건 보고서 구조 정리**: 기본정보, 사건 관계인, 사건 요약, 판정 요약으로 정리하고, 적용법 표기를 `구민법 / 79년 민법 / 현행민법`으로 통일했습니다.
- **상속권 없음 표시 개선**: 상속권 없는 사람은 그래프 노드에서는 숨기되, 보고서에는 raw 사건 관계인과 엔진 판정 결과를 병합해 표시하도록 개선했습니다.
- **지분 표기 정리**: 엣지는 사건 내부 분배분수를 같은 사건 형제 그룹 기준으로 통분하고, 노드 카드는 형제 그룹 기준 통분지분, 보고서 표는 분수 표기로 분리했습니다.
- **가계도 요약 패널 정리**: 가계도 요약 패널은 데이터 입력 탭의 좌측 패널로 두고, 패널 내부에서 닫기/열기 흐름을 처리하도록 정리했습니다.

### ✅ 버그 수정
- **CSV 내보내기 빈 파일 문제 수정**: 지분 요약 내보내기(CSV)가 항상 빈 파일로 저장되던 버그 수정 — `finalShares.survivors` 잘못된 필드 참조를 `direct + subGroups` 병합으로 교체
- **`result` 탭 참조 오류 수정**: 삭제된 `result` 탭을 `PrintReport`, `App.jsx` 등 여러 곳에서 여전히 참조하던 dead code 제거
- **`useSmartGuide.js` 한글 인코딩 깨짐 수정**: 코드 내 `'이름 미상'` 등 문자열이 깨진 채로 저장되어 UI에 표시되던 문제 수정
- **사망일 유효성 검증 추가**: 존재하지 않는 날짜(예: `2024-02-30`, 미완성 입력) 입력 시 빨간 테두리와 "올바른 날짜가 아닙니다" 툴팁 표시, 저장 차단
- **순환 참조 실시간 감지**: 상속인을 자기 자신의 하위에 연결하는 등 순환 구조 발생 시 편집 중 즉시 필수 가이드로 표시 (기존: 계산 시점에만 감지)

### 🆕 신규 기능
- **배우자별 자녀 중복 인물 감지**: 동일인이 wife/husband 탭에 이중 입력된 경우 `duplicate-person` 경고 가이드 표시 — '확인 (다른 사람)' 버튼으로 완료 처리 가능
- **성명 중복 검사**: 같은 부모 아래 동일 이름 상속인이 2명 이상이면 경고 가이드 — '확인 (다른 사람)' 버튼으로 완료 처리 가능
- **재혼 배우자 자녀 범위 확인 가이드**: 재혼 이력(`remarriageDate`)이 있는 배우자 아래 자녀가 있을 때 자녀 범위 확인 권장 가이드 표시
- **입력 탭 blocking 이슈 배너**: 현재 사건 탭에 계산을 막는 이슈가 있으면 입력 탭 상단에 황색 경고 배너 표시

## [베타 v1.0.0] — 2026-04-22

### 🎉 최초 베타 릴리즈

- **4단계 작업 탭 구성**: 데이터 입력 → 사건 검토(시뮬레이션 그래프) → 계산 상세 → 결과 요약
- **연도별 민법 자동 적용**: 피상속인 사망일 기준으로 1960 / 1979 / 1991년 민법을 자동 판별하여 지분 계산
- **스마트 가이드**: 입력 데이터의 논리 오류 및 법률 쟁점을 실시간으로 감지하고 안내
- **사건 그래프 뷰**: 재상속·대습상속 흐름을 직각 연결선 그래프로 시각화
- **AI 자동 입력 모달**: 자연어 텍스트 → AI → JSON 응답을 붙여넣으면 자동으로 가계도 구성
- **호주상속 처리**: 1991년 이전 사망 사건의 호주 자격 부여 및 호주가산(5할) 지분 계산 지원
- **대습상속 자동 판정**: 피상속인보다 먼저 사망한 상속인의 배우자·자녀에게 대습상속 자동 적용
- **오프라인 단독 HTML 배포**: 인터넷 없이 단일 HTML 파일만으로 완전한 기능 사용 가능
- **Windows 설치파일(NSIS) 배포**: 데스크탑 앱으로 설치 가능
- **다크모드 지원**: 라이트/다크 모드 즉시 전환
- **인쇄/PDF 출력 최적화**: 상세 계산서·요약표를 깔끔한 인쇄 레이아웃으로 출력 가능
- **저장/불러오기**: `.json` 파일로 작업 내용 저장 및 재사용
